# Quantum Notebook

Quantum Notebook is an AI-powered research platform built to help students, researchers, and curious minds organize knowledge, conduct research, and build scientific projects.

## Features

- AI Research Assistant
- Research Dashboard
- Scientific Notes
- Simulations
- Formula Workspace
- Secure Authentication
- Cloud Synchronization

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Supabase
- Vite

> The production source code remains private while this repository showcases the project.
